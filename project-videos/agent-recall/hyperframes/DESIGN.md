# Design

## Style Prompt

Use a precise local-infrastructure proof style that matches Agent Recall: warm paper outside the tool, deep terminal navy inside the proof surface, blue command text, mint receipt signals, and amber health markers. The video should feel like an inspectable engineering artifact, not a generic AI promo. The story stays simple: agents need memory they can inspect and reuse, and Agent Recall turns local archives into receipt-backed memory.

## Colors

- Canvas: `#F2EFE7`
- Paper: `#FFF9EF`
- Ink: `#101A23`
- Muted text: `#61706B`
- Terminal: `#0E1726`
- Terminal chrome: `#1E2A3A`
- Border: `#D9D0C2`
- Command blue: `#84B9FF`
- Receipt mint: `#8EE3C2`
- Health amber: `#E4BE55`
- Alert red: `#D95B4D`

## Typography

- Display: `Georgia`, serif
- Interface/body: `Avenir Next`, `Helvetica Neue`, sans-serif
- Data/code: `SF Mono`, `Menlo`, monospace

## Motion

- Six evidence-tour scenes, `42s` total.
- Use medium-energy push transitions between related proof steps, with blur crossfades for the durable-output reveal and the final wind-down.
- Every scene enters with varied opacity, position, scale, rule, and card reveals.
- Keep the public CLI screenshot readable after entry; do not over-zoom private-looking text.
- Use synthetic cards only for safe proof abstractions: source labels, parser actions, receipts, memory objects, health states, and reviewer path.
- The final `3s` is a settled end beat with the project name and one short result line.

## What NOT to Do

- Do not show private transcript contents, private memory records, raw local JSONL blobs, tokens, cookies, API keys, or secret-bearing command output.
- Do not expose private local paths beyond already-public portfolio-safe examples in the source screenshot.
- Do not make the video a vague AI-memory slogan sequence; each scene must show input, action, output, proof, or reviewer path.
- Do not use neon, purple-blue gradients, or generic dark SaaS styling.
- Do not use jump cuts between scenes.
- Do not crowd the frame with implementation copy; one idea per scene.
