## Privacy Boundary

The composition does not use live Telegram, Discord, Codex thread, database, or local runtime screenshots. All message, route, approval, daemon, and thread visuals are synthetic labels created in HyperFrames. No tokens, raw thread ids, local paths, private messages, user handles, or runtime state are shown.
