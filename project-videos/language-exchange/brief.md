# Project Video Brief

Project: Language Exchange Platform

Video type: MVP walkthrough clip

Target duration: 30 seconds

Primary viewer: Hiring manager or general portfolio viewer

Story mode: Product walkthrough

One-sentence job: Language Exchange Platform turns local language practice from profile browsing into small-group discovery, chat, and coordination.

Viewer takeaway: This is a working MVP for a trust-heavy local product, not just a set of static screens.

Plain story:

- Problem: language learners wanted real practice, but profile swiping and large events made conversation hard.
- Solution: HablaBA focused the MVP on small groups, neighborhood discovery, and coordination.
- Result: the product connected onboarding, discovery, group chat, and cross-platform use closely enough to test the behavior.

Scene arc:

1. Problem: practice was hard to make real.
2. Solution: HablaBA centers small local exchanges.
3. Product flow: explore a nearby exchange.
4. Product flow: coordinate in group chat.
5. Result: discovery, chat, and coordination work as one MVP loop.
6. Ending beat: Language Exchange Platform, local practice ready to test.

Reading-speed constraint:

- Keep each scene to one headline plus one short sentence at most.
- Hold product screenshots long enough for inspection.
- Keep the final beat clean for 2.5-3 seconds.
- Keep stack names, repository details, and implementation terms off-screen.

Assets needed:

- Screenshot: public HablaBA exchange discovery screen.
- Screenshot: public HablaBA group chat coordination screen.
- UI clip: not needed; motion is handled in HyperFrames around static public assets.
- Artifact: native flow rail showing onboarding, discovery, chat, and coordination.
- Diagram/data point: simple local-practice loop, not technical architecture.

Privacy notes:

- Hide: private runtime accounts, real contact details, notifications, tokens, local paths, and non-public user data.
- Safe to show: public portfolio screenshots, synthetic HablaBA examples, public project name, and plain workflow labels.
