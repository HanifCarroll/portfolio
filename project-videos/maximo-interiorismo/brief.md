# Project Video Brief

Project: Maximo Interiorismo

Video type: Visual site scroll loop

Target duration: 18 seconds

Primary viewer: Hiring manager or general portfolio viewer

Story mode: Short demo

One-sentence job: Maximo Interiorismo gives an interior designer a credible portfolio site that moves prospects from reviewing work to starting a conversation.

Viewer takeaway: This is a premium launch-site build where the work stays in front and the inquiry path stays clear.

Plain story:

- Problem: the designer's high-end interior work looked stronger in person than online.
- Solution: a restrained portfolio site let the work carry the pitch.
- Result: prospects could inspect projects, trust the presentation, and move toward inquiry.

Scene arc:

1. Open on the branded hero as a stronger public home.
2. Scroll into the project presentation so the interior work leads.
3. Show the review-to-inquiry path through quiet navigation and proof.
4. Hold an ending beat with the project name and result line.

Reading-speed constraint:

- Keep visible copy to one short phrase per scene.
- Keep the loop silent and readable without narration.
- Reserve the final 2.5-3 seconds for a settled ending beat.

Assets needed:

- Screenshot: public Maximo hero site screenshot.
- Screenshot: public Maximo project detail screenshot.
- Logo: public Maximo logo asset.
- UI clip: not needed; scroll motion is built in HyperFrames around static public assets.

Privacy notes:

- Hide: none; only public portfolio screenshots and public logo are used.
- Safe to show: public Maximo site imagery, brand mark, navigation labels, project presentation, and plain result copy.
