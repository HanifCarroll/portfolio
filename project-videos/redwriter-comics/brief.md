# Project Video Brief

Project: Redwriter Comics

Video type: Visual portfolio scroll loop

Target duration: 16-17 seconds

Primary viewer: Hiring manager or general portfolio viewer

Story mode: Short demo

One-sentence job: Redwriter Comics turns scattered public artwork into one professional portfolio surface for editorial review and commission inquiries.

Viewer takeaway: This is a visual launch-site build that makes a comic artist easier to review and commission.

Plain story:

- Problem: the artist's work was strong, but the public presentation was scattered.
- Solution: a clean portfolio hub organized the work around professional review.
- Result: editors and potential commissioners had a clearer path from artwork to inquiry.

Scene arc:

1. Problem: artwork was scattered.
2. Solution: one professional portfolio hub made review easier.
3. Result: review could move into commission conversations.
4. Ending beat: Redwriter Comics, ready for portfolio review and commissions.

Reading-speed constraint:

- Keep each beat to one short line.
- Use the artwork and site screenshots as the proof; copy should stay secondary.
- Hold the final beat for `3s`.

Assets needed:

- Screenshot: Redwriter Comics hero surface.
- Screenshot: Redwriter Comics portfolio feature surface.
- Logo: public Redwriter logo.
- UI clip: not needed; HyperFrames creates the scroll and path motion from static public assets.

Privacy notes:

- Hide: private client correspondence, CMS/admin surfaces, account data, tokens, cookies, and local paths.
- Safe to show: public portfolio screenshots, public logo, project name, and plain portfolio-review labels.
